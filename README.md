# Superstore SQL Analysis Project

This project contains exploratory SQL analysis on the Superstore retail dataset.  
The goal of this project is to demonstrate SQL skills including cleaning, filtering, grouping, aggregation, and extracting business insights.

## Files in this project

- **superstore.db**  
  SQLite database containing the Superstore dataset.

- **queries.sql**  
  SQL queries used to explore the dataset and answer business questions.

## Key Insights From Queries

### 1. Preview of dataset  
Checked the first 10 rows to understand structure.

### 2. Total rows count  
Verified dataset size (9994 rows).

### 3. Distinct categories  
The dataset contains 3 product categories.

### 4. Total sales by category  
Identified which category generated the highest revenue.

### 5. Total sales in 2014  
Converted dates (text format) and aggregated yearly sales.

### 6. Top 10 customers by spend  
Found the highest-value repeat customers.

### 7. Top 10 products by sales  
Highlighted the best-selling products across the dataset.

### 8. Regional profit analysis  
Compared profit performance across regions.

### 9. Monthly sales trend  
Extracted months and years from text-formatted date column.

### 10. Most profitable sub-categories  
Ranked sub-categories by profitability.
